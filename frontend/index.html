使用說明

1. 創建資料夾

點擊「創建資料夾」按鈕，輸入名稱即可創建新的媒體資料夾。

2. 添加媒體

在資料夾中添加 Instagram 貼文、Reels 或 IGTV 的連結。

支援的連結格式：

  • https://www.instagram.com/p/xxxxx/
  • https://www.instagram.com/reel/xxxxx/
  • https://www.instagram.com/tv/xxxxx/

3. 嵌入到網站

複製生成的嵌入代碼，貼到你的網站 HTML 中即可。

4. 自訂樣式

可以調整佈局、間距、顏色等設定，打造符合品牌風格的展示效果。
